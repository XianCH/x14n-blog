    # 命令备忘录

    # 1. 解压文件

    # tar.gz

    解压	： 	tar zxvf FileName.tar.gz
压缩	：	tar zcvf FileName.tar.gz DirName

    # zip

    解压

    unzip

    # tar

    tar –xvf file.tar

    # 2. docker

    停用全部运行中的容器

    docker stop $(docker ps -q)

    .删除全部容器

    docker rm $(docker ps -aq)

    停用并删除容器

    docker stop $(docker ps -q) & docker rm $(docker ps -aq)

    删除所有镜像

    docker rmi $(docker images -q)

    进入容器

    docker exec -it contaionName /bin/bash

    # docker-compose

    构建容器

    docker-compose up -d --build

    # 查看linux下的cpu、内存和硬盘大小：

    查看cpu的方法

    1、 cat /proc/cpuinfo 或者 更直观的查看cpu的型号命令：dmesg |grep -i xeon

    查看内存的方法

    2、 cat /proc/meminfo 或者 更直观的查看内存的命令：free -m

    查看硬盘大小

    3、df -h

    最后用top命令也可以查看到cpu和内存的使用率 在输入top命令之后直接按"1" 就能很清楚的查看到cpu和内存的使用情况。

    # vim快捷键操作

    1，删除命令

    dd 删除当前行

    D删除当前字符到尾行

    2，拷贝和黏贴

    yy 拷贝当前行

    nyy 拷贝当前后开始的 n 行，比如 2yy 拷贝当前行及其下一行。

    3，撤销和重做

    u 撤销（Undo）
    U 撤销对整行的操作
    Ctrl + r 重做（Redo），即撤销的撤销。

    # 安装java jdk

    yum search jdk

    yum install java-1.8.0-openjdk.x86_64

    java -version

    yum安装的默认路径为：/usr/lib/jvm

    jdk的安装路径加入到JAVA_HOME

    vim /etc/profile

    在文件最后加上

    #set java environment
JAVA_HOME=/usr/lib/jvm/jre-1.6.0-openjdk.x86_64
PATH=$PATH:$JAVA_HOME/bin
CLASSPATH=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar
export JAVA_HOME CLASSPATH PATH

    最后

    source /etc/profile
